# Project documentation

Welcome to the project documentation.

This folder centralizes the project's documentation and archived reports.


Contents
- `api/` — Endpoint references by resource (auth, products, orders).
- `setup/` — Environment and installation guides (local-dev, database).
- `architecture/` — High-level architecture notes and flow charts.
- `archive/` — Archived reports and audit files.

Key docs
- `api/auth.md` — Signup and login flows
- `api/products.md` — Catalog and search API
- `api/orders.md` — Checkout and order management
- `setup/local-dev.md` — How to run the project locally
- `setup/database.md` — Migration and schema notes
- `architecture/flow-charts.md` — User journeys and flow diagrams
- `archive/API_PROFESSIONAL_GUIDE.md` — Full API reference and examples (archived copy)


How to use
1. Start the backend (see `shopkart-e-commerce-webside-backend/README.md`).
2. Open Swagger UI: `http://localhost:5000/docs`.
3. Read `archive/API_PROFESSIONAL_GUIDE.md` for API usage and examples.

If you prefer a specific archived report restored to the repository root, tell me which one and I will move it back.
